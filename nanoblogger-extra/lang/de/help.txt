NanoBlogger - Konsolen Weblog Engine.
Version $VERSION, von Kevin Wood <n1xt3r@fastmail.fm>

Benutzung:
 $BASENAME [-b blog_verzeichnis] [Optionen]

Optionen:
 -a, --add				erzeugt einen neuen Eintrag, Kategorie, oder Weblog
					(Verzeichnis).
 -b, --blogdir <Verzeichnis>		spezifiziert das Weblog Verzeichnis.
 -c, --category	<ID,cat>		Spezifiziert die Kategorie (für '--add',
					'--delete', '--edit', '--list',
					'--update', und '--update-cache').
 --cfgfile <file>			Spezifiziert eine alternative Konfigurationsdatei.
 --configure				Konfiguriere Weblog (für '--update').
 --datadir <Verzeichnis>		Spezifiziert das Datenverzeichnis des Weblog.
 -d, --delete <ID,cat>			löscht einen Eintrag oder eine Kategorie.
 -D, --desc <text>			setzt den  Beschreibungstext (für '--add',
 					'--draft', '--makefile' und
 					'--makepage').
 -e, --edit <ID,cat>			Editiert einen Eintrag oder eine Kategorie.
 -E, --draft <file> 			Editiert oder erzeugt eine Metadatei.
 -f, --file <file> 			Spezifiziert ein Metafile zum
					importieren als Eintrag
 					(für '--add').
 -h, --help				Zeigt diese Meldung an.
 -i, --interactive <1=on/0=off>		startet interaktiv (für '--add', '--delete',
 					'--preview', '--publish', und '--update').
 -l, --list <all,cat,DATE,max>		zeigt Einträge oder Kategorien an (Standard ist max).
 --manual				Zeigt das Handbuch an.
 -m, --move <ID>			Verschiebt einen Eintrag in die angegebene
					Kategorie (für '--category').
 -M, --makepage <file1> <file2> 	Erzeugt eine neue Weblog Seite (file2) von
					Metafile (file1).
 --makefile <file> 			Erzeugt ein neues Metafile.
 -n, --author <text>			Setzt den Text für Autor (für '--add',
					'--draft', '--makefile' und
 					'--makepage').
 --plugindir <Verzeichnis> 		Spezifiziert das Plugin Verzeichnis des Weblog.
 -p, --preview				Startet das Kommando für die Weblog-Vorschau.
 -P, --publish				Startet das Kommando zum Veröffentlichen des Weblog.
 -q, --query <DATE>			Spezifiziert ein query zum startet (für '--edit' und
 					'--delete').
 --template <file>			Spezifiert file to load as template (für
 					'--draft', '--makepage' und '--makefile').
 --template-dir <Verzeichnis>		Spezifiziert das Weblog Template-Verzeichnis.
 -t, --title <text>			Setzt den Text des Titels (für '--add',
					'--category', '--draft',
 					'--makefile' und '--makepage').
 --tag <TAG> 	 	 		Specifiziert ein TAG (für '--add', '--draft',
					'--edit', '--makefile' und '--makepage').
 --tag-text <text> 			Setzt den Text des TAG (für '--tag').
 -T, --text <text> 			Setzt Text des Body (für '--add',
					'--draft', '--makefile' und '--makepage').
 -U, --update-cache <all,DATE,max,expired>
 					Erzwingt eine Aktualisierung des Weblog Cache 
					und entfernt die angegebenen gecachten Daten
					(Standardwert ist 'expired').
 -u, --update <all,DATE,main,max> 	Erzwingt eine Aktualisierung der Weblog-Dateien, 
					wobei die angegebenen Dateien aktualisiert werden
					(Standardwert ist 'max').
 -v, --verbose <1=on/0=off>		Umschalten zwischen langen und kurzen Meldungen.
 -V, --version				Zeigt die Versionsinformation an.

 -c,-d,-e,-m akzeptieren mehrere IDs durch Komma getrennt (z.B. 1,2,3).
 
 --list und --update können auf ein Datum beschränkt werden (irrelevant bei Eintrag ID(s)).

 DATE = CCYY-MM-DD, CCYY-MM, CCYY (oder ein Zeitraum = DATE1,DATE2)

Beispiele:

 Spezifiziert das Weblog Verzeichnis, das erzeugt werden soll.
	nb -b ~/public_html/weblog -a

 Erzeugt eine neue Kategorie mit dem Titel "News"
	nb -b ~/public_html/weblog -t "News" -c new -a

 Erzeugt einen neuen Eintrag ausschließlich mit Kommandozeilenoptionen!
	nb -b ~/public_html/weblog -n MeinName -t 'MeinTitel' -T 'Text!' -a
