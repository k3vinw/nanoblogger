Usage:
 $BASENAME [-b blog_dir] [options] <action>

Actions:
 add <entry,tag,weblog> 		- create new entry, tag, or weblog
					  (directory).
 configure				- configure weblog (for 'update').
 delete <entry,tag> <ID>		- delete an entry, tag, or un-tag entry.
 draft <FILE> 				- edit or create metafile.
 edit <entry,tag> <ID>			- edit an entry or tag.
 list					- list entries or tags (for
 					  'query' and 'tag', defaults to
					  'max').
 make-page <FILE1> <FILE2> 		- create new weblog page (file2) from
					  metafile (file1).
 tag-entry <ID>				- tag entry (for 'tag').
 tag-list 				- list tags (same as 'query tag list').
 manual					- view the manual.
 make-file <FILE> 			- create new metafile.
 preview				- run command to preview weblog.
 publish				- run command to publish weblog.
 query <all,tag,DATE,main,max, 		- specify a query to run (for 'edit',
        expired,articles,feeds> 	  'delete', 'list', 'tag-entry',
					  'update', and 'update-cache').
 tag <ID,new> 				- specify tag (for 'add', 'delete',
 					  'edit', 'list', 'update', and
					  'update-cache').
 file <FILE> 				- specify metafile to import as entry
 					  (for 'add').
 update-cache 				- force cache update (for 'query',
 					  defaults to 'expired').
 update 				- force weblog update (for 'query',
 					  defaults to 'max').

Options:
 -b, --blog-dir <DIRECTORY>		specify weblog directory.
 -f, --force 				force cache update. (synonymous with
 					'update-cache').
 -h,--help				show this help message.
 -i, --interactive <1=on/0=off> 	toggle interactive mode (for 'add',
 					'delete', 'preview', 'publish', and
					'update').
 -v, --verbose <1=on/0=off>		toggle verbosity.
 --conf-file <FILE>			specify an alternate configuration file.
 --desc <text>				set text of description (for 'add',
 					'draft', 'make-file' and 'make-page').
 --data-dir <DIRECTORY>			specify weblog's data directory.
 --title <TEXT> 			set title (for 'add', 'tag', 'draft',
 					'make-file' and 'make-page').
 --author <TEXT>			set text of author (for 'add', 'draft',
 					'make-file' and 'make-page').
 --plugin-dir <DIRECTORY> 		specify weblog's plugin directory.
 --text <TEXT> 				set text of body (for 'add', 'draft',
 					'make-file' and 'make-page').
 --template <FILE>			specify file to load as template (for
 					'draft', 'make-page' and 'make-file').
 --template-dir <DIRECTORY>		specify weblog's template directory.
 --var <VARIABLE>  	 		define meta-variable (for 'add',
 					'draft', 'edit', 'make-file' and
					'make-page').
 --set-var <INPUT>  			set meta-variable (for '--var').
 --version				display version information.

 tag,delete,edit,tag-entry accepts multiple ID numbers separated by commas (e.g.
 1,2,3).
 
 DATE = CCYY-MM-DD, CCYY-MM, CCYY (or range = DATE1,DATE2)

note: special queries such as "articles" and "feeds" only take effect when a
      plugin is loaded that can interpret the query and are typically combined
      with the "update" action.

Examples:

 create new weblog using specified non-existant directory
	nb -b ~/public_html/weblog add weblog

 create new "examples" tag
	nb -b ~/public_html/weblog --title examples add tag

 create new entry using nothing but command line options
	nb -b ~/public_html/weblog --author 'myname' --title 'Title' --text 'Text!' add entry
